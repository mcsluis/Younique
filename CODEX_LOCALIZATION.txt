YOUNIQUE - ENGELSE LOCALIZATION INSTRUCTIE VOOR CODEX
=====================================================
Doel:        De volledige iOS-app Younique vertalen van Nederlands naar
             Engels via Apple's String Catalog. Nederlands blijft de
             default; Engels is de tweede locale.
Voorkennis:  Lees eerst CLAUDE_HANDOFF.txt en CODEX_BRIEFING.txt voor
             app-context, architectuur en conventies.
Geschat werk: 2-3 uur, ~150 strings, ~17 Swift bestanden, 1 HTML.

================================================
1. AANPAK
================================================

Gebruik een **String Catalog** (.xcstrings) — Apple's moderne aanpak. Een
enkel bestand, GUI in Xcode, automatische change-detection.

Stap A: nieuw bestand maken
  File -> New File -> String Catalog -> name: "Localizable.xcstrings"
  Plaats in:  Younique/Younique/Localizable.xcstrings

Stap B: Engels toevoegen
  Open de .xcstrings file in Xcode -> klik "+" linksonder -> kies English
  (Nederlands staat al als development language).

Stap C: alle Text("..."), NSLocalizedString, en LocalizedStringKey strings
  in code laten "ontdekken" door een build te draaien. Xcode pakt ze
  automatisch op zolang je geinterpoleerde strings zoals "Hallo \(naam)"
  vervangt door String(localized: "Hallo \(naam)") of LocalizedStringKey
  patronen.

Stap D: voor elke ontdekte key handmatig de Engelse vertaling invoeren in
  het String Catalog scherm.

4. TONE-GUIDE VOOR ENGELS
================================================

- Gewone, vriendelijke US English. NIET Britse spelling.
- Direct en helder, niet pompeus. De NL-tekst is informeel ("jullie"),
  Engels gebruikt "you" (zonder onderscheid).
- Modal verbs spaarzaam ("you can ... ", niet "you might be able to ... ").
- Microcopy in zinnen, geen alles-met-hoofdletters.
- Knop-labels in title case: "Discover a Name", "Restore Purchases",
  "Unlock Premium for $X", "Done", "Close", "Save".
- Section headers in sentence case: "Name type", "Number of positions",
  "Sound style".

Voorbeelden:
- "Vind jullie unieke naam" -> "Find your one-of-a-kind name"
- "Ontdek een naam" -> "Discover a Name"
- "Haal meer uit Younique" -> "Get more out of Younique"
- "Altijd gratis" -> "Always free"
- "Met Premium krijg je erbij" -> "Premium adds"
- "Veilig betalen via de App Store." -> "Secure payment via the App Store."
- "Klankstijl" -> "Sound style"
- "Naamtype" -> "Name type"
- "Geavanceerd aanpassen" -> "Advanced settings"

================================================
5. STAPPENPLAN
================================================

1. CONFIGURATIE (~10 min)
   - Open project in Xcode
   - File -> New File -> String Catalog -> name "Localizable.xcstrings"
   - Plaats in Younique/Younique/
   - Open het bestand -> + onderaan -> kies English
   - Set "nl" als source language indien niet automatisch

2. CODE-CONVERSIE (~1-1.5 uur)
   - Loop bestanden A1-A10, dan B1-B5, dan C1-C3 in deze volgorde.
   - In SwiftUI Text("..."): wordt automatisch gepakt als
     LocalizedStringKey. Geen wijziging nodig in code zelf -- alleen het
     String Catalog vult zich met de keys bij build.
   - Voor String-properties (zoals NameType.title):
       return String(localized: "Girl name")
     in plaats van return "Meisjesnaam".
   - Voor geinterpoleerde strings (`"Positie \(x): \(y) geselecteerd"`):
       return String(localized: "Position \(x): \(y) selected")
     -- de placeholder %lld of %@ wordt automatisch gegenereerd.
   - Voor accessibility-strings: .accessibilityLabel(LocalizedStringKey("..."))
     of .accessibilityLabel(String(localized: "..."))
   - Belangrijk: BUILD na elk bestand om Xcode strings te laten opdoen.

3. VERTALEN (~45-60 min)
   - Open Localizable.xcstrings in Xcode -> tab "English"
   - Loop alle keys door, vul Engelse vertaling.
   - Gebruik strikt de terminologie uit sectie 3.

4. PRIVACY HTML (~10 min)
   - Kopieer docs/privacy.html naar docs/privacy-en.html
   - Vertaal alle <p> en <h2> teksten.
   - Voeg in beide HTML's onderaan een mini lang-switcher toe:
     <a href="privacy-en.html">English</a> | <a href="privacy.html">Nederlands</a>

5. TESTEN (~15 min)
   - Build de app
   - Run in simulator
   - Test in NL (default): alles moet identiek zijn
   - Wissel simulator naar Engels:
     Settings -> General -> Language & Region -> iPhone Language -> English
   - Doorloop hele app: home, paywall, settings, info, onboarding,
     shortlist, favorite detail, privacy.
   - Controleer dat geen Nederlandse strings overblijven.
   - Run RunAllTests om regressie te checken.

================================================
6. EDGE CASES / VALKUILEN
================================================

A. **Lange titels in Menu**: sommige SyllableSelectionMode-titles zijn al
   lang in NL ("Gekozen lettergrepen beschikbaar volgens de
   positiestructuur"). Engelse versie moet ook in de Menu passen.
   Suggestie: hou Engelse versie even compact of compacter.

B. **Sample-syllables in SoundStylePreset**: de syllables zelf (la, mi,
   bel, etc.) blijven onveranderd in beide talen -- die staan al lowercase
   en zijn taal-agnostisch.

C. **Plurals**: gebruik String Catalog's plural feature voor zinnen als
   "1 klankgroep verborgen" / "X klankgroepen verborgen".

D. **Datumformaat in FavoriteDetailView**: `favorite.savedAt.formatted()`
   gebruikt automatisch de locale-instelling. Geen aanpassing nodig.

E. **Configuration.storekit**: heeft al NL+EN localizations voor het
   product. Niet aanraken.

F. **App naam "Younique"**: NIET vertalen. Blijft Younique overal.

G. **Onboarding**: belangrijke conversie-stap, vertaal vooral zorgvuldig.

H. **NameProfile.summary**: kan geinterpoleerde tekst bevatten. Open
   eerst om te zien wat er staat.
