1. WAT IS DE APP
================================================

Younique is een SwiftUI iOS-app voor (aanstaande) ouders om unieke baby-
voornamen te genereren door zorgvuldig samengestelde lettergrepen te
combineren via een slot-machine UI.

Doelgroep: Nederlandstalige aanstaande ouders.
Platform: iOS 26.5+ (iPhone + iPad universal).
Architectuur: SwiftUI + SwiftData + StoreKit 2 + @Observable ViewModels.
Geen Combine, geen UIKit-fallbacks. Async/await waar nodig.

5. BUILD & TEST
================================================

Build via Xcode (Cmd+B) of via MCP tool BuildProject.
Tests draaien via Cmd+U of RunAllTests.
StoreKit testen: Edit Scheme → Run → StoreKit Configuration →
"Configuration.storekit". Daarna kun je in Debug → StoreKit → Manage
Transactions de aankopen resetten.

================================================
6. CONVENTIES & STIJL
================================================

- SwiftUI views: PascalCase, conform to View, body als computed.
- @State private var voor lokale state.
- @Environment(...) voor injected services (PurchaseManager).
- @AppStorage voor persistente prefs (displayFont).
- 4-space indentatie.
- Geen Combine. Async/await voor alles.
- Comments alleen bij niet-obvious logica.
- Tests via Swift Testing framework (niet XCTest).
- UI tests via XCUIAutomation.
- Dutch language voor user-facing strings; English voor code identifiers.
