WORKFLOW SIMULATOR
------------------
1. Boot simulator:    iPhone 16 Pro Max (voor 6.9")
                      iPhone 15 Plus    (voor 6.7")
2. Set device language: Settings → General → Language & Region.
   Voor "nl" schermen: device op Nederlands.
   Voor "en" schermen: device op Engels.
3. Build & run Younique vanaf Xcode op die simulator.
4. Zet de app in de juiste state voor de shot.
5. Cmd+S in simulator → standaard naar ~/Desktop. Verplaats naar de
   juiste subdir met de juiste naam.

ALTERNATIEF: in-app taalwissel via Settings → Taal (overrides systeemtaal).
Handig als je niet steeds de simulator-taal wilt switchen.
