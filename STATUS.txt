BEKENDE CAVEATS
===================================================================
- pbxproj: developmentRegion=nl, knownRegions=nl,en,Base. Bundle ID =
  com.mcs.younique (app target). Niet zonder akkoord aanpassen.
- IAP product ID is ÉÉN bron van waarheid: code (PurchaseManager
  unlockProductID) + Configuration.storekit + ASC moeten exact gelijk
  zijn: com.mcs.younique.unlock_full.
- PurchaseManager.freeLaunch (PurchaseManager.swift:18) is de master-
  schakelaar voor gratis launch. true = alle premium-features open + UI
  verborgen. false = normaal gedrag (IAP gate aan). Eén regel flippen,
  geen UI-werk nodig om Premium weer aan te zetten.
- StoreKit Configuration in scheme = ALLEEN Run (debug). Productie
  TestFlight/App Store ignoreert dit en praat met ASC.
- iCloud container = iCloud.com.mcs.younique (na de bundle ID switch).
  iCloud-sync is NIET code-gated achter Premium; werkt voor iedereen
  met iCloud aan, ongeacht freeLaunch.
- 116 stale entries in Localizable.xcstrings zijn grotendeels false
  positives (worden runtime gebruikt via Bundle.appLocalizedString die
  de extractor niet ziet). Bewust laten staan; bij twijfel grep eerst.
- "Wat vind je van de naam %@?", "Ontgrendel Premium voor %@" en
  "Laden..." staan op extractionState: manual om auto-stale te
  voorkomen.
- Liquid Glass toolbar pills renderen automatisch in fullScreenCover
  ZONDER navigationTitle. Workaround = handmatige Capsule.
- UI tests gebruiken accessibility identifiers (NIET tekstmatch). Bij
  refactor van knoppen: identifier behouden ("discoverButton",
  "spinAgainButton").
- translations.csv staat in .gitignore — gegenereerd bestand.
- Codex (parallel agent) commit soms grotere refactors — altijd `git
  diff` reviewen voor commit-summaries.
